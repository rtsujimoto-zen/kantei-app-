# 十二大従星の解説と言い換え

## 対応表

* 天報星：(001 オンリーワン)
* 天印星：(108 安心安全)
* 天貴星：(919 マネ学ぶ力)
* 天恍星：(888 氣品)
* 天南星：(012 情報)
* 天禄星：(100 完璧)
* 天将星：(555 頭角)
* 天堂星：(125 境界の力)
* 天胡星：(789 化ける)
* 天庫星：(025 和を大切にする力)
* 天馳星：(000 自由)
* 天極星：(024 トコトン)

## 解説

* **天報星 (001 オンリーワン)**
    * **一般的な意味:** 受胎したばかりの「胎児」のエネルギーを持つ星です。多芸多才で好奇心旺盛、変化の多い環境で輝きますが、器用貧乏になりやすい側面もあります。
    * **あなたの資質:**
        * **特徴:** 独創的で、人と同じを好まない性質があります。
        * **エネルギーのイメージ:** 過去から現在、そして未来のアイデアへと、一点に集中して進んでいくエネルギーです。
        * **生かし方:** 「ひとりの時間こそエネルギーチャージ」の時です。 孤独を愛するからこそ、独自の感性や才能が磨かれます。
        * **殺し方:** この星は「寂しがり屋の束縛嫌い」です。 他人に合わせすぎると、個性が失われます。
        * **極まった状態:** 誰にも真似できない「独特の感性・才能」を発揮する存在になります。

* **天印星 (108 安心安全)**
    * **一般的な意味:** 生まれたての「赤ちゃん」のエネルギーを持つ星です。誰からも愛され、助けられる徳を持ちますが、依存心が強くなることもあります。
    * **あなたの資質:**
        * **特徴:** 慎重で、些細なことで不安や心配を感じやすいです。
        * **エネルギーのイメージ:** 好きか嫌いか、白黒はっきりした価値観の軸を持っています。
        * **生かし方:** 「心がワクワクするかどうか」が重要です。 好奇心旺盛で大胆な一面も持っており、いざという時には肝が据わります。
        * **殺し方:** ワクワクしない単調な環境にいると、持ち前の無邪気さが失われます。
        * **極まった状態:** 純粋で「無邪気」な魅力を放ち、周囲を和ませる存在になります。

* **天貴星 (919 マネ学ぶ力)**
    * **一般的な意味:** 素直で真面目な「小学生」のエネルギーを持つ星です。学習能力が高く、プライドも高いですが、打たれ弱い一面も持ちます。
    * **あなたの資質:**
        * **特徴:** 学ぶとは「真似る」ことだと知っており、見て学ぶ能力が抜群です。
        * **エネルギーのイメージ:** 物事の要点や結論を素早く掴み、全体を把握します。
        * **生かし方:** 「要点や結論が大事」で、物事を伝える力、他者を巻き込む力があります。
        * **殺し方:** じっと考えすぎてしまい、時に「先走り」や「迷い」が生じることがあります。
        * **極まった状態:** 優れた「伝える力・他者を巻き込む力」を持つリーダーになります。

* **天恍星 (888 氣品)**
    * **一般的な意味:** 華やかで夢見がちな「思春期の中高生」のエネルギーを持つ星です。芸術性や社交性に優れますが、現実逃避しやすい傾向もあります。
    * **あなたの資質:**
        * **特徴:** 努力を見せない美学があり、変化こそが宝となります。
        * **エネルギーのイメージ:** 変化、多様性、拡大、ダイナミックさがエネルギーの源です。
        * **生かし方:** 「チャレンジすることから能力が発揮」されます。 飛躍のエネルギーを持ち、海外にも縁があります。
        * **殺し方:** 「仕事や手法を固定化するとエネルギーが低下」してしまいます。
        * **極まった状態:** 「飛躍のエネルギー」を使いこなし、常に新しいことに挑戦するカリスマとなります。

* **天南星 (012 情報)**
    * **一般的な意味:** エネルギッシュで怖いもの知らずな「青年」のエネルギーを持つ星です。行動力と批判精神が旺盛ですが、無鉄砲で反発心が強くなることもあります。
    * **あなたの資質:**
        * **特徴:** ボランティア精神にあふれ、情報を先取りする力があります。
        * **エネルギーのイメージ:** スタートダッシュに強く、最初に全力を注ぎます。
        * **生かし方:** 「入手した情報をみんなに分け与える」ことで、信頼と人望を集めます。
        * **殺し方:** 争いを嫌い、本音を言わない傾向があります。 情報を古いままで止めると魅力が半減します。
        * **極まった状態:** 「予感、予知夢」といった直感力に優れ、時代の流れを先読みします。

* **天禄星 (100 完璧)**
    * **一般的な意味:** 経験と安定感のある「壮年・家長」のエネルギーを持つ星です。堅実で信頼が厚いですが、用心深く冒険を嫌う面もあります。
    * **あなたの資質:**
        * **特徴:** 「職人気質」で、物事を0から100まで完璧にやり遂げます。
        * **エネルギーのイメージ:** 常に高みを目指し、山を越えていくようなエネルギーです。
        * **生かし方:** 「まだまだもっと！」と自己評価を安定させていくことで、どこまでも成長できます。
        * **殺し方:** 「甘えない、甘え方が分からずいつも忙しい」状態に陥り、自信を喪失することがあります。
        * **極まった状態:** 人を導く「最高の指導者」となります。

* **天将星 (555 頭角)**
    * **一般的な意味:** 最もエネルギーが強い「王様・頂点」の星です。リーダーシップと独立心がありますが、ワンマンで人の意見を聞かない頑固さも持ち合わせます。
    * **あなたの資質:**
        * **特徴:** 全体把握とバランス感覚に優れ、上から目線を嫌います。
        * **エネルギーのイメージ:** 即断即決はせず、中心でどっしりと構えるエネルギーです。
        * **生かし方:** 「公平・平等をとても大切に」します。 大事なことであればあるほど、自分なりの時間をかけて結論を出します。
        * **殺し方:** 急な決断を迫られたり、不公平な状況に置かれたりすると、力を発揮できません。
        * **極まった状態:** 圧倒的な「存在感」を放つリーダーとなります。

* **天堂星 (125 境界の力)**
    * **一般的な意味:** 引退した「老人」のエネルギーを持つ星です。冷静で知恵があり、達観していますが、若々しい行動力には欠け、理屈っぽくなることがあります。
    * **あなたの資質:**
        * **特徴:** 精神的な心を大切に歩み、二面性を持っています。
        * **エネルギーのイメージ:** 見えるモノと見えないモノ、陰と陽のバランスを取るエネルギーです。
        * **生かし方:** 「ボーっとすることと寝ることがエネルギーチャージ」になります。 私たちはどこから来てどこへ行くのか、といった根源的な問いに惹かれます。
        * **殺し方:** 常に忙しく、物理的な世界だけに囚われると、精神的なバランスを崩します。
        * **極まった状態:** 長期的な視点で物事を捉える「長期の力」を発揮します。

* **天胡星 (789 化ける)**
    * **一般的な意味:** 「病人」のエネルギーを持つ星です。感受性が鋭く、直感力や芸術的才能に恵まれますが、繊細で疲れやすい側面があります。
    * **あなたの資質:**
        * **特徴:** 人生で年上がキーマンとなり、経験を大切に考えます。
        * **エネルギーのイメージ:** 経験A、経験Bと積み重ねていくことで、真理にたどり着きます。
        * **生かし方:** 「経験からウソや本物を見抜く眼が宿り」ます。
        * **殺し方:** 「見せかけの格好良さに心が動かなく」なります。 表面的なものに惑わされると、本質を見失います。
        * **極まった状態:** あらゆる経験を統合し、物事の「真理」を掴む賢者となります。

* **天極星 (024 トコトン)**
    * **一般的な意味:** 「死人」のエネルギーを持つ星です。精神世界に関心が強く、お人好しですが、現実的な欲が薄く、物事に執着しないため流されやすいことがあります。
    * **あなたの資質:**
        * **特徴:** 「どうぞこの人生、好きなことで歩かれてくださいませ」という星です。
        * **エネルギーのイメージ:** 好きなことには集中し、そうでないものは聞こえない・見えないほど一途です。
        * **生かし方:** 「好きなことにしか能力が発揮できない」ため、好きなことを追求することが最も重要です。 「音なき音を聞き取る」ような繊細な感覚を持ちます。
        * **殺し方:** 興味のないことを無理に続けると、すべての能力が発揮できなくなります。
        * **極まった状態:** 類まれな「気配り」能力や、声なき声を聞き取る力を持ちます。

* **天庫星 (025 和を大切にする力)**
    * **一般的な意味:** 「墓に入った状態」のエネルギーを持つ星です。探求心や凝り性で、集中力に優れますが、一族や家系に縛られやすい傾向があります。
    * **あなたの資質:**
        * **特徴:** 和を乱す人を嫌う「究極の寂しがり屋」です。
        * **エネルギーのイメージ:** 人と人との繋がりを中心に、波紋が広がるようなエネルギーです。
        * **生かし方:** 「世のため、人のためになるような事をしたい」という精神性を持っています。
        * **殺し方:** 「ずっと大勢の中にいるととても疲れる」ため、一人の時間も必要です。
        * **極まった状態:** 優れた「分析力・客観性」を発揮し、グループの調和を保ちます。

* **天馳星 (000 自由)**
    * **一般的な意味:** 「あの世」のエネルギーを持つ星です。瞬発力と多忙さを宿命とし、目まぐるしく動き回ることで力を発揮しますが、落ち着きがなく継続性に欠ける面があります。
    * **あなたの資質:**
        * **特徴:** 「瞬間の大器」であり、物事をギリギリでこなす才能があります。
        * **エネルギーのイメージ:** 一直線ではなく、自由な軌道を描きながらゴールに向かうエネルギーです。
        * **生かし方:** 「自由こそがエネルギー源」です。 「羽の中に才能あり」と言われ、ひらめきやアイデアを大切にすることで才能が花開きます。
        * **殺し方:** 「束縛されると能力が発揮できなく」なります。
        * **極まった状態:** 枠にとらわれない優れた「ひらめき・アイデア」を生み出す天才になります。
